Install ugettext:
     pip3 install python-gettext

Gettext must be installed on windows. Use this link: https://mlocati.github.io/articles/gettext-iconv-windows.html

For each app:
    Go to the app folder:
        cd app_name
    If not exists, create locale folder inside the app:
        mkdir locale
    Create .po files:
        For each language you want, run:
            python ../manage.py makemessages -l {language_code} --verbosity 2 -i migrations  --settings devstack
            # language_code: fr, en, ar / --verbosity: for showing details / -i: for ignoring files and folders / --settings for settings file.
        If .po files not appearing in project structure and showed in unresolved files, add them by git:
            git add {path/to/app}/locale/{language}/LC_MESSAGES/django.po
    Create .mo files by running:
        python manage.py compilemessages  --settings devstack
